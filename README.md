# Projeto Banco de Dados Loja

Projeto desenvolvido para a disciplina **Programação e Desenvolvimento de Banco de Dados**, com o objetivo de criar, popular e consultar uma base de dados relacional utilizando MySQL.

## Estrutura dos Scripts

- **diagrama.sql:** Criação do banco de dados `Loja` e suas tabelas (Estado, Município, Cliente, ContaReceber).
- **inserir.sql:** Inserção de registros em todas as tabelas.
- **consulta.sql:** Criação da view `ContasNaoPagas` para listar contas com situação = 1.

## Ferramentas Utilizadas
- MySQL Server
- MySQL Workbench
